Our App's Amazing Features

What is it?

GMajor is a collaborative mobile application that lets friends engage in musical conversation.

How does it work?

Using the experimental web audio API and synthesizer technology, users can build tracks with a variety of instruments.

What's up with the grid?

The grid-based user interface, that is pre-loaded with scales, makes it easy to build great sounding tracks.
